## 想法
只挑利潤是正的就好，會先設10001是因為第一天沒利潤，然後prices最高為10000，所以設10001。

```CPP
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int sum = 0 ;
        int last = 10001;
        for(int i = 0 ; i < prices.size() ; i++)
        {
            if(prices[i] - last > 0)
            {
                sum = sum + prices[i] - last;
            }
            last = prices[i];
        }
        
        return sum;
    }
};
```
