## 想法
low跟high分別代表前一個數為波谷時的最長長度與波峰時的最常長度。

如果當前數字比前一個數字大的話，那high會是low加上1。

如果當前數字比前一個數字小的話，那low會是high加上1。

```CPP
class Solution {
public:
    int wiggleMaxLength(vector<int>& nums) {
        int low = 1, high = 1;
        for(int i = 1 ; i < nums.size() ; i++)
        {
            if(nums[i] > nums[i - 1])
            {
                high = low + 1;
            }
            else if(nums[i] < nums[i - 1])
            {
                low = high + 1;
            }
        }

        return max(low, high);
    }
};
```
